# 🎨 Mondrian Painting — CSS Grid Project

A pixel-accurate recreation of Piet Mondrian's iconic painting built using only **HTML** and **CSS Grid** — no hints, no AI assistance, no guidance. Built entirely by looking at a photo of the original painting and replicating it from scratch.

## 🛠️ Tech Stack

- HTML5
- CSS3 (Grid Layout)

## 🎯 Why I Built This

This project was built immediately after completing the CSS Grid section of my full stack web development journey. The goal was to test whether I could independently translate a real-world visual composition into a working CSS Grid layout — without any scaffolding or help. Mondrian's paintings are a classic CSS Grid exercise because the layout maps directly to rows, columns, and nested grids.

## 📌 What I Practiced

- CSS Grid (`grid-template-columns`, `grid-template-rows`, `gap`)
- Nested grids for complex sub-layouts
- Pixel-level sizing and spacing
- Translating a 2D visual composition into a DOM structure

## ⚠️ Known Limitations

| Issue | Detail |
|---|---|
| ID-based styling only | All styling targets IDs, which limits reusability |
| `#d1b` has duplicate CSS rules | `background-color: #F9D01E` is overridden later by the grid declaration — works but is redundant |
